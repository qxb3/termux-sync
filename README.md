# termux-sync

A simple bash script that syncs up two project without worrying about symlinks errors.

## Installation

```bash
git clone https://github.com/qxb3/termux-sync.git
cd termux-sync
bash install.sh
```

## Usage

```bash
termux-sync projects/my-awsome-website/ storage/shared/websites/
```
